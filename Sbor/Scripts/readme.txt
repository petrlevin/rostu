Скрипты миграции для БД. Поддерживается только upgrade, downgrade не предусмотрен.

Номер скрипта должен быть больше или равен номеру ревизии, на которой в репозиторий была отправлена структура БД, ожидаемая данным скриптом.
Например:

1137 - добавили поле
1137.sql - скрипт заполняющий данное поле значениями.
